# Zeconic Web Solutions

Marketing website for Zeconic Web Solutions — a sub-brand of Zeconic LLC.

Built with Next.js 14, Tailwind CSS, Framer Motion, GSAP, and React Three Fiber.

## Stack

- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion + GSAP ScrollTrigger
- 3D: React Three Fiber + Three.js
- Deployment: Vercel

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push to main → Vercel auto-deploys.

Add `NEXT_PUBLIC_SITE_URL` to Vercel environment variables.

## Screenshots

Add project screenshots to `/public/screenshots/` for portfolio cards.
